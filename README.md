# Meet Jack (RU) - Персональный лендинг

Русская версия персонального лендинга с 3D-персонажем Максом.

## Особенности

- 3D-сцена с Spline
- Адаптивный дизайн
- Темная тема
- Кириллические шрифты (IBM Plex Mono, Montserrat)
- Vite + React + TypeScript + Tailwind CSS

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Стек

- Vite
- React 18
- TypeScript
- Tailwind CSS
- Spline 3D
- shadcn/ui

## Структура

```
src/
  components/
    Header.tsx
    Footer.tsx
    SplineScene.tsx
    HeroTextOverlay.tsx
    RotatingTextAccent.tsx
  pages/
    Index.tsx
```
